# Overview
##### Look:
The game is going to be low poly 3D passed through a pixel art shader. The game will also have a hex grid.
##### Size and proportions:
Size and proportions are going to be relatively realistic and humanoid. Similar to the game BitCraft. Not a lot of stylistic exaggerations.
##### Genre and feel:
Genre is an open world fantasy that is somewhat post apocalyptic. The feel is going to be grim-hope/hope-core. Similar vibes to Project Hail Mary, except the world is already destroyed. The game feel should be similar to a dark fairytale type of game - as in, it's dark and post apocalyptic but it isn't inappropriate for children but it's still interesting enough for teens/adults. Hollow knight is a good example of this.

I'm not sure what I want the apocalyptic event to be, but I definitely don't want it to make the world look uniform and barren and boring. I want it to feel lived in. Maybe the apocalyptic event is that the world was merged with another world, or that the world destroyed everything that wasn't magical or wasn't protected by something magical. I'm not sure. There is a lot of freedom in this area.
