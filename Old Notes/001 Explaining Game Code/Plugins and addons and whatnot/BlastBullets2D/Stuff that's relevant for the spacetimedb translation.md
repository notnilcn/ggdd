DirectionalBullets vs BlockBullets
Directional bullets home in on a moving target. BlockBullets do not.

Speed and rotation curves
These are literally just 2d graphs where x is the time and y is the speed/rotation. You can just send an array of x,y coordinates.

Movement patterns
These are a little more complicated. They are nodes that need to be loaded. You can probably reconstruct the nodes as variables and pass them as data into the set_movement_pattern function.
