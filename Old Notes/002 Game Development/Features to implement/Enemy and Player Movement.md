Enemy and player movement are going to be similar.

The only difference is that enemies will have movement patterns. What this means is that instead of lerping in a straight line from A to B, the enemies lerp along a path. As in, if an enemy's movement pattern is "move in a circle" then it'll lerp around in a circle.

Enemies will also be able to chase players. This is what the "target: EnemyTarget" column is supposed to be for.
